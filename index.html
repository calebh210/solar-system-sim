Places Near Me

An API integrated web app which shows a user points of interest near them on a map once they enter their location. When a point of interest in selected, a description of the location and a picture of the location is presented on the page.

The project was created for an assignment in Web Authoring. This was a group project, and my main contributions were creating the User Interface, and interfacing the Google Maps API
